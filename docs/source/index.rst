Welcome to windwatts_data's documentation!
=======================

**windwatts_data** is a Python library built to facilitate easy access to the climate data such as WTK-LED 1224 data.
This package enables users to fetch and analyze the data in realtime. It pulls the data from an public repository on S3(AWS Cloud Storage) called **wtk-led**
and offers simple python functions to perform desired operations on the data.

.. note::
   This project is under active development.
